# Spatial-Data-Structures-k-d-Tree-Search-
&emsp;**Консольное приложение на языке C для работы с пространственными данными**<br>
&emsp;[Установка](#установка) • [Использование](#использование)
</div>
Библиотека для работы с многомерными пространственными данными, реализующая эффективные алгоритмы поиска и кластеризации. Включает построение k-мерного дерева поиска (k-d tree), алгоритмы кластеризации Fuzzy C-means и DBSCAN.

## Функциональность

*   **k-d дерево (k-d tree)**
    *   Вставка новой точки
    *   Поиск ближайшего соседа
*   **Fuzzy C-means (FCM)**
    *   Мягкая кластеризация точек с вычислением степеней принадлежности
*   **DBSCAN**
    *   Кластеризация на основе плотности с выделением шума

## Структура
*   main.c - Основная программа и интерфейс командной строки
*   kdtree.c / .h - Реализация k-d дерева
*   fcm.c / .h - Алгоритм Fuzzy C-means
*   dbscan.c / .h - Алгоритм DBSCAN
*   sample.csv - Пример тестовых данных

## Установка

### Требования

- **Компилятор C**: GCC или Clang 
- **ОС**: Linux, macOS, Windows (MinGW/Cygwin)
- **Библиотеки**: стандартная библиотека C

### Компиляция

#### Базовая компиляция
```bash
gcc main.c kdtree.c fcm.c dbscan.c -o robot -lm
```
*   Флаг -lm используется для линковки математической библиотеки.
*   После успешной компиляции будет создан исполняемый файл robot (или robot.exe в Windows)

## Использование
*   Общий синтаксис:
```bash
./robot <путь_к_csv_файлу> <команда> [параметры]
```
## Поддерживаемые команды

*   **-kd_insert**
    *   Назначение: Добавление новой точки в k-d дерево.
    *   ```bash
        ./robot <файл.csv> -kd_insert x,y,z,...
        ```
*   **-kd_nearest**
    *   Назначение: Поиск ближайшего соседа к заданной точке с использованием k-d дерева.
    *   ```bash
        ./robot <файл.csv> -kd_nearest x,y,z,...
        ```
*   **-cmeans**
    *   Назначение: Запуск алгоритма нечёткой кластеризации Fuzzy C-means (C-средних).
    *   ```bash
        ./robot <файл.csv> -cmeans N
        ```
*   **-dbscan**
    *   Назначение: Запуск алгоритма кластеризации DBSCAN
    *   ```bash
        ./robot <файл.csv> -dbscan eps,minPts
        ```
## Формат входных данных
*   CSV-файл должен соответствовать следующим требованиям:
    *   Разделитель: запятая (,)
    *   Содержимое: только числовые значения (целые или с плавающей запятой)
    *   Размерность: одинаковая для всех точек
